# Программа в C#

## Максимальное и минимальное число

После ввода "число 1" и "число 2" выводит на экран, какое из них наибольшее, а какое наименьшее.

Введите число 1: 8

Введите число 2: 2

Наибольшее число 1: 8

Наименьшее число 2: 2
